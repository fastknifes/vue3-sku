# 规格组图片（外部组件）

说明：开启“规格组图片”后，可为规格组的子项上传图片，并在联动模式下映射到 SKU 行。

关键点：

  • showGroupImageSwitch: 显示每个规格组的图片开关
  • imageComponent: 传入外部上传组件（v-model 绑定图片 URL）
  • resolveImageProps(scope): 向上传组件注入必要的属性（如分组信息）
  • groupImageSwitchTip: 开关提示默认文案为“开启后可为该规格组的子项上传图片”

示例代码：

<template>
  <SkuForm
    v-model:attribute="attribute"
    v-model:sku="sku"
    :source-attribute="sourceAttribute"
    :show-group-image-switch="true"
    :image-component="MyUploader"
    :resolve-image-props="resolveImageProps"
  />
</template>

<script>
import MyUploader from '@/components/MyUploader.vue'
export default {
  components: { MyUploader },
  data() {
    return {
      sourceAttribute: [
        { name: '颜色', item: ['黑', '金', '白'], enableImage: true },
        { name: '内存', item: ['16G', '32G'] }
      ],
      attribute: [],
      sku: []
    }
  },
  methods: {
    resolveImageProps(scope) {
      return { group: scope.groupName, item: scope.item.name }
    }
  }
}
</script>

注意：需传入实际的上传组件，该组件以 v-model 双向绑定图片地址。
