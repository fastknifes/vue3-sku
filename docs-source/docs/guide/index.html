# 快速上手

使用注意

因为 vue-sku-form 基于 ElementUI 开发，安装使用前确保项目中已正确安装 ElementUI

# 推荐使用 yarn 进行安装
yarn add vue-sku-form
# 或者使用 npm 也无伤大雅
npm install vue-sku-form

安装完成后，可以选择全局引入或者局部引入。全局引入需要在 main.js 中增加以下内容：

import SkuForm from 'vue-sku-form'
Vue.use(SkuForm)

然后在页面中就可以使用 <SkuForm /> 组件了，例如：

<template>
    <SkuForm />
    <!-- 这里为示例代码片段，文档站不直接渲染运行 -->

局部引入则在页面中手动引入：

<template>
    <SkuForm />
</template>

<script>
import SkuForm from 'vue-sku-form'

export default {
	components: {
		SkuForm
	}
}
</script>
